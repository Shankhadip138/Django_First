Pushing
-------------------------
1- git init
2- git add .
3- git commit -m "Add all my files"
4- git remote add origin https://github.com/USER_NAME/FOLDER_NAME
5- git pull origin master --allow-unrelated-histories
6- git push origin master


coping repository
---------------------------
* git clone "URL for Git" (if new)

* git pull (if already exist and need to take the new uploaded files, make sure you are in the same directory and no changes were made in the local repo)
